Chapter 01
이동 안전
🚌 버스 탑승과 하차
사고는 목적지에서만 나는 게 아닙니다.
출발부터 도착까지 전 과정이 교육활동입니다.
01 이동 안전
버스 안에서의 약속
1
탑승 즉시 안전벨트 착용 '딸깍' 소리가 날 때까지. 어깨·골반선에 위치. 급정거 시 안전벨트 없으면 앞좌석에 얼굴이 충돌합니다.
2
이동 중 통로 이동·장난 금지 버스가 움직이는 동안 자리 이동은 전도 사고로 이어집니다. 통로는 비상 탈출 경로.
3
하차 시 도로 확인 후 이동 차에서 내리면 즉시 인도로. 절대 뛰지 않기. 반대편 차량 먼저 확인.
4
선생님 지시 전 독단 이동 절대 금지 현장 도착 후 "잠깐 저기만요"가 사고의 시작입니다. 항상 2인 이상, 선생님께 먼저 보고.

02 텐트 생활
텐트 안전 수칙
✅ 해야 할 것
  • 설치 전 평탄·배수 방향 확인
  • 취침 전 환기 유지 (출입구 일부 개방)
  • 야간 이동은 반드시 2인 이상
  • 랜턴 사용, 라이터·성냥 소지 금지
⛔ 절대 금지
  • 텐트 안 버너·모기향 사용 → 일산화탄소 중독
  • 텐트 안 음식 보관 → 야생동물 유인
  • 야간 무단 단독 이탈
🚧 남녀 구역 분리 운영
  • 상대 성별 구역에 허락 없이 출입하는 것은 학교폭력
  • 화장실 이동 → 같은 성별 2인 이상 동행
  • 불편한 일 → 문자로도 괜찮으니 즉시 선생님께
💬
신고가 자신을 지키는 가장 용기 있는 행동입니다.
🤒

끙끙대다가 더 커질 수 있습니다.

🍳 요리도 교육입니다

03-A 버너 안전
버너는 어른과 함께 반드시
✕
학생 단독 점화·소화 절대 금지 가스 연결·점화·끄기 모두 지도교사 입회 하에만.
✕
텐트 근처·밀폐 공간 버너 사용 금지 일산화탄소(CO)는 무색·무취. 증상 없이 의식을 잃습니다.
1
사용 후 반드시 가스 밸브 잠금 두 번 확인 잠갔다고 생각했는데 안 잠긴 경우가 대부분입니다.
2
조리 구역에서만 사용 · 주변 낙엽·풀 사전 제거 바람 방향 확인 후 점화. 사용 후 불씨 완전 소화 확인.
🍚 취사는 교육의 일부입니다
  • 밥 짓기·반찬 만들기는 직접 해봐야 배웁니다
  • 조별로 역할 분담 후 다 함께 참여
  • 간편식(즉석식품)에만 의존하면 배울 것이 없습니다
  • 설거지·뒷정리까지가 취사 교육입니다
🔥 화재 발생 시
  • "불이야!" 크게 외쳐 주변 알리기
  • 작은 불이어도 혼자 끄려 하지 말 것
  • 연기를 등지고 낮은 자세로 대피
  • 대피 후 선생님 보고
03-B 고기 굽기 주의
덜 익은 고기는 야영을 망칩니다
⚠️ 왜 위험한가
  • 덜 익은 고기 → 캠필로박터·살모넬라균
  • 증상: 구토·설사·복통 → 야영 중 최악
  • 탈수 동반 → 응급 이송 가능
  • 특히 돼지고기·닭고기는 속까지 반드시
🍖
두꺼운 삼겹살 사온 조는 특히 주의!
겉은 타도 속이 생고기인 경우가 많습니다.
✅ 고기 굽기 체크리스트

    두꺼우면 칼로 잘라서 단면 직접 확인
  • 단면 색깔 → 분홍색 = 덜 익음
    회갈색으로 완전히 변해야 안전
  • 익힌 고기·생고기 집게 분리 사용
  • 먹다 남은 고기 장시간 상온 방치 금지
    여름철 2시간 이상 방치 → 폐기

05 기상·체온 안전
알면 살 수 있습니다
☀️ 열사병·열탈진
증상: 체온 급상승, 두통, 어지럼증, 구역질, 의식 혼탁
  • 즉시 그늘·서늘한 곳으로 이동
  • 옷 느슨하게 → 목·겨드랑이·사타구니 찬 물수건
  • 의식 있으면 물 천천히 마시게 하기
  • 의식 없으면 즉시 선생님께 알리기
❄️ 저체온증
증상: 심한 떨림, 말이 어눌, 움직임 둔화, 의식 저하
  • 즉시 따뜻한 장소로 이동
  • 젖은 옷 제거 → 담요·침낭으로 전신 보온
  • 따뜻한 음료 (의식 있을 때만)
  • 마사지·알코올 금지 → 선생님 즉시 알리기
💧
수분 보충: 30분마다 물 한 컵, 하루 최소 1.5L 갈증이 없어도 마십니다. 에너지음료·탄산은 오히려 탈수를 악화시킵니다.
⚡
번개 발생 시 나무·높은 곳에서 즉시 멀어지기 낮은 자세로 이동. 혼자 고립 절대 금지. 선생님 지시에 따라 신속 대피.

즉시 알리세요.
1
담임 선생님·수련지도사에게 즉시 알리기 가장 먼저. 상황·위치·증상 설명.
2
캠핑장 관리소 연락 (선생님이 진행) 캠핑장 이름 + 구역 번호 파악해두기.
3
119 신고 (선생님이 진행) 위치·인원·의식 여부 설명. 끊지 말고 지시 따르기.
🩹 기본 응급처치
  • 출혈·상처: 깨끗한 천으로 눌러 지혈. 깊은 상처는 선생님 즉시 보고.
  • 골절 의심: 절대 움직이지 말고 고정. 함부로 잡아당기지 않기.
  • 발목 삐었을 때: RICE — 휴식·얼음찜질·압박·들어올리기.
  • 구토·복통(식중독): 수분 보충하면서 즉시 선생님께 보고.
  • 의식 없음: 반응 확인 → 없으면 선생님 즉시 호출 + CPR.
